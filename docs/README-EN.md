# **AliPlayerKit Documentation Index**

**AliPlayerKit** is a business-oriented set of **low-code player UI components** and **scenario-based solutions**. By encapsulating player capabilities and UI interactions at a high level, it helps clients quickly build in-app video playback with minimal integration effort — no need to call low-level player APIs directly or implement complex player UI yourself.

This documentation index describes the content and purpose of each document, organized according to the recommended learning and usage flow.
